clean - эта команда очищает target( очистка артефактов)
validate - проверка все ли корректно (pom и тд)
compile - это компиляция java кода в .class (проверка ошибок в коде)
test - запускает unit тесты
package - упаковывает в архив .jar (.var)
verify - проверка все ли тесты (unit) прошли успешно
install - установка приложения (артефакта) в локальный репозиторий
site - генерация документа (java docs)
deploy - отправка готового артефакта на удаленный репозиторий

mvn dependency:tree просмотр дерево зависимостей
